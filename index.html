My Projects

Clove Dental

I built a responsive Clove Dental website that showcases treatments, doctors, and clinics with simple appointment booking and clear calls-to-action. It focuses on trust, usability, and performance across devices, and is developed usingHTML5, CSS3, and JavaScript with a lightweight CMS backend

LEO Fashion Store

I developed a responsive e-commerce Product Listing Page using React.js and CSS. It integrates the FakeStore API with filtering, sorting, and search, while maintaining strong SEO, Accessibility, and Performance. Deployed on Netlify with clean, modular React Components.

Products Showcase

I built a full-stack Product Showcase & Enquiry System using React, Node.js / Express, and SQLite. Users can browse products, view details, and submit enquiries, while admins manage all submissions. The system includes search, filters, pagination, and secure REST APIs with a clean, modular architecture.

Resume Maker

I developed a dynamic Resume Maker application using React.js (earlier HTML/JS version), with data storage via PostgreSQL and Node.js. Users can build resumes with editable sections, live preview, multiple templates, and export to PDF using html2pdf.js. It supports dynamic fields, autosave, and real-time validation with a clean, user-friendly interface.

Expense Tracker

I built a modern Expense Tracker application using React with Vite for fast builds, styled with Tailwind CSS and shadcn/ui, along with Radix UI components. Interactive charts are powered by Recharts, icons by Lucide, and the app is configured as a PWA with proper Open Graph metadata for sharing. The result is a responsive, fast, and user-friendly finance dashboard for tracking expenses and insights.

Sports Facility

I developed a full-stack Sports Facility Booking Platform using React, Node.js / Express, and MongoDB. It supports secure JWT authentication. The platform includes secure APIs, role-based access, and a clean modular architecture. Admin Dashboard with charts powered by RechartsThe platform includes secure APIs, role-based access, and a clean modular architecture.
